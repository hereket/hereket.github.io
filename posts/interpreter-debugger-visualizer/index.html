Intro

Linkg to the demo: Interpereter visualizer

I had some play around with dart desctop and web code and for that I wanted to build non trivial application to stress test to see if it will be able to handle a bit more complex project. For quite some time I had an idea to build a visualizer for a programming language and it seemed like a project with appropriate difficulty curve for the task in mind.

For this project there were a few possible languages. Some were elimated due to their complexity and some are too dynamic. Lua were a good candidate but it felt a bit off. Here we will be using a toy language named Lox. It’s grammar is stable and fixed, it covers most important core topics in current popular languages and most imporatantly there is a book by Robert Nystrom that describes how to build your exact copy if you want.

I tried to keep language exactly as it is described in the book with code very similar to what is Robert did in his version. Here I will share the grammar and a little of description of how it all fits together.

For now I will leave it here. This grammar is just to share with my friends infromation about the languag and how to use it. Later I will add more information to this article

Grammar

program        → declartion* EOF ;
declaration    → classDecl
               | funDecl
               | varDecl
               | statement ;
classDecl      → "class" IDENTIFIER ( "<" IDENTIFIER )?
                 "{" function* "}" ;
funDecl        → "fun" function ;
function       → IDENTIFIER "(" parameters? ")" block ;
parameters     → IDENTIFIER ( "," IDENTIFIER )* ;
statement      → exprStmt
               | forStmt
               | ifStmt
               | printStmt
               | returnStmt
               | whileStmt
               | block ;
returnStmt     → "return" expression? ";" ; 
forStmt        → "for" "(" ( varDecl | exprStmt | ";" )
                 expression? ";"
                 expression? ")" statement ;
whileStmt      → "while" "(" expression ")" statement ;
ifStmt         → "if" "(" expression ")" statement
               ( "else" statement )? ;
block          → "{" declaration* "}" ;
exprStmt       → expression ";" ;
printStmt      → "print" expression ";" ;
varDecl        → "var" IDENTIFIER ( "=" expression )? ";" ;



expression     → assignment ;
assignment     → ( call "." )? IDENTIFIER "=" assignment
               | logic_or ;
logic_or       → logic_and ( "or" logic_and )* ;
logic_and      → equality ( "and" equality )* ;
equality       → comparison ( ( "!=" | "==" ) comparison )* ;
comparison     → term ( ( ">" | ">=" | "<" | "<=" ) term )* ;
term           → factor ( ( "-" | "+" ) factor )* ;
factor         → unary ( ( "/" | "*" ) unary )* ;
unary          → ( "!" | "-" ) unary | call ;
call           → primary ( "(" arguments? ")" | "." IDENTIFIER )* ;
arguments      → expression ( "," expression )* ;
primary        → "true" | "false" | "nil" | "this"
               | NUMBER | STRING | IDENTIFIER | "(" expression ")"
               | "super" "." IDENTIFIER ;
